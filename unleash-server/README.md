# unleash-server
unleash-server is a place to ask for the status of features.

# Important commands:

```
// Set up DB
cp config/database.example.json config/database.json
npm run db-setup

// Start server in dev-mode:
npm run start-dev

// Admin dashboard
http://localhost:4242

// Feature API:
http://localhost:4242/features

// Execute tests:
npm test
```
